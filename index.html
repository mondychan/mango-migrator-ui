Mango Migrator UI

Import klientů z ISPAdminu do Mango přes CSV nebo ISPAdmin API.

Last run: —

Postup

  1. Vyber zdroj dat: původní CSV nebo nový ISPAdmin API.
  2. Zdroj nejdřív připrav: u CSV nahraj oba soubory, u API vytvoř snapshot aktivních i deaktivovaných klientů.
  3. Spusť job. Sync nejdřív deaktivuje klienty z active=0, potom doplní klienty z active=1, kteří v Mango ještě nejsou.
Aplikace vždy importuje do Mango postupně, aby zbytečně nepřetěžovala API. Připravený zdroj se po doběhu smaže.

Zatím není připraven žádný zdroj.

CSV režim

Použije stávající workflow se soubory deaktivovani-klienti.csv a kontakty.csv.

CSV režim zachovává i deaktivace.

ISPAdmin API režim

Backend načte aktivní i deaktivované klienty z ISPAdmin REST API pomocí tokenu a URL nastavených v cibs.env.

API sync: nejdřív porovná Mango s klienty active=0 a provede deaktivace, potom z klientů active=1 doplní jen chybějící záznamy.
Vytvoří lokální snapshot aktivních i deaktivovaných klientů pro aktuální běh.
